# CookEye — Fogao Inteligente com Interface Integrada

Demonstracao interativa do **CookEye**, um fogao inteligente com interface digital integrada. O site funciona como um showcase premium onde o usuario explora as 8 funcionalidades do produto clicando nos icones posicionados sobre a imagem do fogao.

## Funcionalidades

| # | Recurso | Descricao |
|---|---------|-----------|
| 1 | Receitas com Video | Biblioteca de receitas com videos passo a passo |
| 2 | Modo de Seguranca | Alertas e desligamento automatico em situacoes de risco |
| 3 | Cozimento Automatico | Ajuste de calor e tempo baseado na receita |
| 4 | Area de IA | Sugestoes de receitas com ingredientes disponiveis |
| 5 | Integracao com Celular | Notificacoes e monitoramento remoto |
| 6 | Acessibilidade por Voz | Comandos de voz e retorno em audio |
| 7 | Modo de Diagnostico | Verificacao de status e manutencao preventiva |
| 8 | Liga/Desliga | Controle central e desligamento de emergencia |

## Stack

- **React 19** + **TypeScript**
- **Vite** (build tool)
- **Tailwind CSS** (estilizacao)
- **Framer Motion** (animacoes)
- **React Router** (navegacao)
- **Lucide React** (icones)

## Como rodar

```bash
npm install
npm run dev
```

## Estrutura

```
src/
  components/
    stove/       # StoveHero, StoveHotspot, FeatureGrid
    ui/          # AppShell, Header, Footer

  pages/         # HomePage, FeaturePage, NotFoundPage
```
